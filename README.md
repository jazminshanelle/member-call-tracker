# Member Call Tracker
A python-based call tracking application inspired by healthcare member service workflows.

## Overview

Member Call Tracker is a Python application designed to simulate a basic call tracking workflow used in customer service and healthcare member support environments.

The program collects information about a member interaction, including the member ID, call type, call duration, and whether the issue was resolved. It then generates a structured call summary.

## Why I Built This Project

I built this project to practice Python programming while creating something related to my professional experience in Member Solutions.

Rather than building a generic beginner project, I wanted to create a tool inspired by real-world call center workflows and customer support operations.

## Example Workflow

1. Enter Member ID
2. Select call type
3. Enter call duration
4. Indicate whether the issue was resolved
5. Generate a call summary
